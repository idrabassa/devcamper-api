# DevCamper API

> Backend API for DevCamper Applications which is a bootcamp directory website 

## Usage

Rename "config/config.env.env" to config/config.env and update the values/settings to your own

## Install Dependencies

```
npm install
```

## Run App
```
#run in dev mode
npm run dev

#run in prod mode
npm start
```
